# Kurdish Kurmanji (Latin) Keyboard for Windows 11

This is a custom keyboard layout for typing Kurmanji Kurdish using the Latin (Hawar) alphabet on Windows 11. 

## How to Install:
1. Download the `.zip` file from this repository.
2. Extract the folder to your computer.
3. Open the folder and double-click `setup.exe`.
4. Restart your PC.

## How to Type Kurmanji Letters:
Switch to this keyboard layout using `Windows Key + Spacebar`. Hold down the **Right Alt key (AltGr)** and press:
* **AltGr + C** = ç (Shift for Ç)
* **AltGr + E** = ê (Shift for Ê)
* **AltGr + I** = î (Shift for Î)
* **AltGr + S** = ş (Shift for Ş)
* **AltGr + U** = û (Shift for Û)
